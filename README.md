# Page 11 Entry Generator

Generate properly formatted NAVPERS 1070/613 (Page 11/Administrative Remarks) entries for common situations.

## Overview

This tool helps NCOs, SNCOs, Officers, and Admin personnel create properly formatted Page 11 entries. It includes templates for:

- **Training & Qualifications** - Rifle/Pistol qualification, PFT/CFT, PME, Formal schools
- **Counselings (Non-Adverse)** - Initial counseling, Mid-marking period, Career counseling
- **Adverse Counselings** - Performance deficiency, Conduct deficiency
- **6105 Counseling** - With UCMJ article dropdown per IRAM Para 4005.8
- **Administrative Entries** - BAH certification, Motorcycle/Tattoo acknowledgments, BCP assignment
- **Recognition** - Letters of appreciation, Awards
- **Custom Entry** - Build your own entry format

## Features

- Pre-written templates following MCO P1070.12K (IRAM) Chapter 4
- Military date formatting (DD Mon YYYY)
- Real-time preview
- Copy to clipboard
- Print-friendly output
- Save drafts locally
- Works offline (PWA)
- Light/Dark/Night mode themes

## Usage

1. Select a category and entry type
2. Set the entry date (defaults to today)
3. Fill in the template fields
4. Preview and copy your entry

## References

- MCO P1070.12K (Individual Records Administration Manual - IRAM) - Chapter 4
- MILPERSMAN 1070-320 (Navy Page 13 guidance)
- NAVPERS 1070/613 form

## Installation

### As a Web App

Visit: https://jeranaias.github.io/page11-generator/

### Install as PWA

1. Open the site in Chrome/Edge/Safari
2. Click "Install" or "Add to Home Screen"
3. Use offline anytime

### Run Locally

```bash
# Clone the repository
git clone https://github.com/jeranaias/page11-generator.git

# Open index.html in your browser
# Or serve with any static file server
npx serve .
```
